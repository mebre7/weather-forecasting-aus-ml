# Australia Rainfall Prediction using Logistic Regression

## 🌦️ Project Overview
This project uses the "Rain in Australia" dataset to predict whether it will rain tomorrow based on meteorological observations from various stations across the country. 

The goal is to build a binary classification model using **Logistic Regression** to estimate the probability of precipitation.

## 📊 Dataset
The dataset contains approximately 10 years of daily weather observations. Key features include:
* **Temperature:** Min/Max temp.
* **Humidity:** Recorded at 9am and 3pm.
* **Pressure:** Atmospheric pressure.
* **Wind:** Speed and direction.
* **Target:** `RainTomorrow` (Yes/No).
